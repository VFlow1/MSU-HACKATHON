## 🛡️ คำถามที่กรรมการอาจถาม (พร้อมคำตอบสำรอง)

### คำถามที่ 1: "AI ใช้ API Key ฟรีหรือเปล่า? ถ้าหมดโควต้าล่ะ?"

> **"ดีครับ — ระบบเรามี AI Router 4 ชั้น ถ้า Groq หมดโควต้า จะสลับไป OpenRouter ฟรี ถ้าหมดอีกจะสลับไป Gemini ฟรี Tier แล้วถ้าทั้งหมดล่มจริง ๆ ระบบจะใช้ Local Rule-based ที่วิเคราะห์จากฐานข้อมูลจริงในตัว ไม่มีวันที่ระบบเงียบครับ"**

### คำถามที่ 2: "ข้อมูลเก็บไว้ไหน? ปลอดภัยไหม?"

> **"ขณะนี้ใช้ LocalStorage บนเบราว์เซอร์ผู้ใช้ ไม่มีข้อมูลส่งออกไป Server ใด ๆ ทั้งสิ้น นอกจากการเรียก AI API ที่ส่งเฉพาะข้อความ Prompt ไปเท่านั้น ไม่มีข้อมูลส่วนตัวพนักงานหลุดออกไปครับ"**

### คำถามที่ 3: "ทำไมต้องเป็น PWA? ต่างจากเว็บธรรมดายังไง?"

> **"เพราะผู้จัดการโครงการต้องดู Dashboard ตลอดเวลา แม้จะอยู่ในที่ที่ไม่มี Wi-Fi หรือเน็ตหลุดระหว่างประชุม PWA ทำให้ติดตั้งเป็นแอปบนมือถือได้ ใช้งานลื่นไหลเหมือน Native App แต่ไม่ต้องผ่าน App Store ครับ"**

### คำถามที่ 4: "AI Agent สร้างงานแล้ว ถ้าผู้จัดการไม่เห็นด้วยล่ะ?"

> **"AI Agent จะสร้างเป็น 'ร่าง' ให้ดูก่อนเสมอ ผู้จัดการต้องกด 'อนุมัติ' ถึงจะสร้างงานจริง ไม่มีการสร้างงานอัตโนมัติโดยไม่มีมนุษย์ตรวจสอบครับ"**

### คำถามที่ 5: "ระบบนี้ Scale ได้ไหม? ถ้ามีพนักงาน 1,000 คน?"

> **"ปัจจุบันเป็น Client-side App ที่เหมาะกับทีมขนาดกลาง ถ้าต้องการ Scale ขึ้น แผน Phase 2 คือย้าย LocalStorage เป็น IndexedDB แล้วต่อ Backend API ที่รองรับ Pagination และ Search แบบ Server-side แต่สถาปัตยกรรม Frontend แบบ Modular ที่เราสร้างไว้ จะย้ายไปต่อ Backend ได้ง่ายมากครับ"**
